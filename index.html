Marauder (Bruce Firmware Build)

My first cybersecurity build: a portable, battery-powered Marauder using a CYD touchscreen ESP32 running Bruce firmware. It can scan nearby Wi-Fi networks, run evil portal demonstrations (shown below), scan Bluetooth devices, emulate a Bluetooth keyboard, and perform Bluetooth spam testing, plus several other wireless security testing features. The whole thing needs no soldering, so every component can be reused for future builds and experiments.

End-to-End Encrypted Group Chat

Real-time encrypted chat for up to 8 users, with peer-to-peer WebRTC voice. Built on Node.js and Socket.io using the browser's native Web Crypto API, no crypto libraries. Per-pair ECDH key exchange with HKDF ratchets gives every message a fresh key, and the server only ever relays ciphertext.
